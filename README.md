## About

Aplikasi E-Docs yang digunakan untuk melakukan pengarsipan dan pencatatan surat masuk dan surat keluar untuk Bada Pengelolaan Pajak dan Retribusi Daerah Barito Kuala Kalimantan Selatan

### Requirements
- PHP 7.4
- Laravel 8.*
- Bootstrap 4.*
- Stisla Admin Template
- Database MySQL 
- Laravel/Dompdf
- Simple-QR Code
- Laravel/ui
